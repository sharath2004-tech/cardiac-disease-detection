9. Federated Learning (Privacy)
Hospitals cannot share patient data.
Federated learning solves this problem.
Traditional training
Hospital A data
Hospital B data
Hospital C data

All data sent to central server
This violates privacy.
________________________________________
Federated learning approach
Hospital A → train model locally
Hospital B → train model locally
Hospital C → train model locally
Only model weights are shared.
Then server aggregates models using:
FedAvg Algorithm
Result:
Global Model
Advantages:
•	Data never leaves hospital
•	Privacy preserved
•	Multi-hospital collaboration possible
