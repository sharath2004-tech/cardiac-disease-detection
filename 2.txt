9. Federated Learning (Privacy)
Hospitals cannot share patient data.
Federated learning allows collaborative training.
Traditional training
Hospital A → data sent to server
Hospital B → data sent to server
Hospital C → data sent to server
Federated training
Hospital A → train locally
Hospital B → train locally
Hospital C → train locally
Only model weights are shared.
Aggregation method:
FedAvg Algorithm
Result:
Global model
Benefits:
•	patient privacy protection
•	multi-hospital collaboration
•	real-world deployment potential
